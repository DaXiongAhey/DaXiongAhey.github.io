每一个人都有自己搭建一个博客的想法，但是要自己编写博客前端后端+服务器太过于麻烦，特别是服务器也是一笔不少的开支。GitHub提供了GitHub Page，其空间完全可以容纳一个博客，并且运用HEXO框架可以尽可能少的占用空间。因为HEXO是静态博客，所以只需要交给GitHub托管就行，并不需要服务器来运算动态部分。
